# Web Academy

## Esqueleto de aplicação com frontend React.

Para rodar a aplicação, execute os seguintes comandos:

```
$ cp .env.example .env
$ cp frontend/.env.example frontend/.env
$ cd frontend && npm install && cd ..
$ docker compose up -d
```
